## Base Layout And Header

## Goal
Create the base layout and header skeleton for the site.

## Inputs
- `russabuss-init-prompt.md`
- `.cursorrules` component patterns and navigation

## Tools/Scripts
- None required.

## Process
1. Check if `src/layouts/BaseLayout.astro` exists and follows the project spec.
2. Create or update `src/layouts/BaseLayout.astro` with a minimal layout shell.
3. Create or update `src/components/astro/Header.astro` with navigation links.
4. Keep markup minimal and accessible.
5. Avoid adding complex logic or animations at this stage.

## Outputs
- `src/layouts/BaseLayout.astro`
- `src/components/astro/Header.astro`

## Edge Cases
- If components already exist, avoid breaking changes and document deltas.
