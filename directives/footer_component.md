## Footer Component

## Goal
Build the site footer component with the required copy.

## Inputs
- `.cursorrules` footer copy section

## Tools/Scripts
- None required.

## Process
1. Create or update `src/components/astro/Footer.astro`.
2. Use the exact footer copy from `.cursorrules`.
3. Keep markup semantic and accessible.
4. Add minimal styles only if needed.
5. Avoid changing unrelated files.

## Outputs
- `src/components/astro/Footer.astro`

## Edge Cases
- If footer exists, preserve existing comments and only update required copy.
