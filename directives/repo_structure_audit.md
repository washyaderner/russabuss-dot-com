## Repo Structure Audit

## Goal
Verify the current repository structure matches `russabuss-init-prompt.md` and list all gaps.

## Inputs
- `russabuss-init-prompt.md`
- Repository tree at repo root, `src/`, and `public/`

## Tools/Scripts
- None required. Use file listing and reading tools only.

## Process
1. Read the folder structure section in `russabuss-init-prompt.md`.
2. Inventory current folders and key files at repo root, `src/`, and `public/`.
3. Compare expected vs actual structure.
4. Produce a clear checklist of missing or mismatched items.
5. Do not modify any source files.

## Outputs
- `.tmp/repo-structure-audit.md` with a checklist and brief notes.

## Edge Cases
- If no gaps are found, note that explicitly.
